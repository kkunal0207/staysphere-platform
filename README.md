# staysphere-platform
StaySphere is a cloud-native accommodation booking platform designed with a modular monolith architecture and later evolved into microservices. It demonstrates real-world backend engineering practices including dynamic pricing, secure role-based access, scheduled workflows, and scalable system design using Spring Boot, Docker, and Kubernetes.

## Project Timeline

### Week 1 – Domain & Architecture Design
- Requirement analysis
- ER diagram design
- Domain boundaries identified
- Inventory & booking strategy finalized

### Week 2 – Domain Modeling (v0.1)
- JPA entities created from ER diagram
- Entity relationships mapped
- Hibernate-based table generation

### Week 3 – Hotel Manager APIs (In Progress)
- Accommodation onboarding APIs
- Room & room-type management
- Inventory setup APIs
